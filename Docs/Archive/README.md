# Archived Development Notes

This directory contains archived development notes, implementation summaries, and PR tracking documents from the Olympe Engine project.

## Purpose

These files are **NOT active documentation** for users or developers. They are historical records of:

- Implementation summaries and technical briefs from specific features
- PR (Pull Request) summaries and tracking notes
- Development plans and test plans from past work
- Architecture exploration documents from early phases

## Active Documentation

For current, maintained documentation, please refer to:

- **Official Documentation Site**: [https://atlasbruce.github.io/Olympe-Engine/](https://atlasbruce.github.io/Olympe-Engine/)
- **API Reference**: [https://atlasbruce.github.io/Olympe-Engine/api](https://atlasbruce.github.io/Olympe-Engine/api)
- **Main Documentation Directory**: [../README.md](../README.md)

This archive contains historical documentation for:

### General Development
- Implementation summaries (old versions)
