# Grid Configuration Guide

Detailed configuration options for the Grid System.

## Grid Settings Panel (F4)

### Basic Settings

- **Grid Enabled**: Toggle grid on/off (also TAB key)
- **Projection Type**: Orthogonal, Isometric, or Hexagonal

### Dimensions

**Ortho/Iso Grids:**
- Width: 8-256 units (default: 32)
- Height: 8-256 units (default: 32)

**Hex Grid:**
- Radius: 4-128 units (default: 16)

### Visual Settings

- **Color**: RGBA picker
- **Max Lines**: 100-5000 (default: 1200)
- **LOD Threshold**: 0.1-2.0 (default: 0.5)
- **LOD Skip Factor**: 1-50 (default: 10)

### Overlays

**Collision Overlay (C key):**
- Enable/disable visualization
- Select layer 0-7

**Navigation Overlay (N key):**
- Enable/disable visualization
- Select layer 0-7

## Quick Actions

- **Reset to Defaults**: Restore all settings
- **Reload from ECS**: Refresh from component data
