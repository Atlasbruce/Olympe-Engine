# Introduction to Olympe Engine

Olympe Engine is a 2D game engine designed with flexibility and performance in mind. Built on Entity Component System (ECS) architecture, it provides a solid foundation for creating 2D games with advanced features.

## What is Olympe Engine?

Olympe Engine combines modern game development practices with ease of use:

- **Entity Component System** - Modular and performant architecture
- **Data-Driven Design** - JSON-based blueprints for rapid iteration
- **Advanced AI** - Built-in behavior trees and pathfinding
- **Map Integration** - Seamless Tiled map editor support

## Who is it for?

Olympe Engine is ideal for:

- Game developers looking for a flexible 2D engine
- Developers wanting to learn ECS architecture
- Projects requiring advanced AI and pathfinding
- Teams needing data-driven game development
