# Olympe Engine Overview

## Current Date
2026-03-14

## Overview
Olympe Engine is a cutting-edge game engine designed for versatility and performance in modern game development. With a modular architecture and innovative features, Olympe empowers developers to create complex gaming experiences efficiently.

## Development Phase
Currently in phase **21-D (Dynamic Pins)**, focusing on enhancing the editor functionalities to support dynamic connections between components.

## Key Features
- **Entity Component System (ECS)**: Provides a flexible system for managing game entities and their behavior.
- **AI System with Behavior Trees**: Robust AI capabilities for creating intelligent non-player characters.
- **Animation Editor**: Powerful tools for creating and managing animations seamlessly.

## Project Status
Olympe Engine is actively in development, with ongoing contributions and enhancements being made to expand its capabilities and performance.

## Key Tools and Editors
- **Blueprint Graph Editor**: A visual scripting tool that allows for intuitive game logic design.
- **3D Model Viewer**: For previewing assets within the development environment.
- **Texture Editor**: Tools for texture manipulation and integration.

## Architecture Overview
Olympe Engine's architecture is built around a highly modular design, enabling easy addition and removal of features with minimal impact on the core system, promoting a scalable development process.
